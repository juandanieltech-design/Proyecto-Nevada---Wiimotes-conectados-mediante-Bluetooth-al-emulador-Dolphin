# Videos - Proyecto Nevada

Esta carpeta contiene evidencia audiovisual del funcionamiento del Nintendo Wii Remote original conectado a Dolphin Emulator.

Los videos incluidos documentan pruebas reales realizadas durante Proyecto Nevada.

## Pruebas documentadas

### New Super Mario Bros Wii

Demostración del funcionamiento del movimiento mediante Wiimote físico.

Comprobado:

- Nintendo Wii Remote RVL-036.
- Movimiento mediante acelerómetro.
- Reconocimiento de gestos.
- Uso de Nunchuk.
- Comunicación funcional con Dolphin Emulator.



---

### Super Mario Galaxy

Demostración del sistema de apuntado infrarrojo.

Comprobado:

- Nintendo Wii Remote RVL-036.
- Sensor Bar Wii U original.
- Cámara infrarroja del Wiimote.
- Puntero funcionando dentro de Dolphin Emulator.
- Configuración final utilizando Wiimote físico con Emulated Wii Remote.
